Garmin Activities Exporter

About

This is a Vercel-hosted service that exports Garmin Connect activities to a JSON file, refreshed daily at 20:00 UTC.

Available Endpoints

Activities Data

Get the latest activities data in JSON format:

/training.json

This endpoint fetches live data from Garmin Connect, including up to 2000 of the most recent activities.

Refresh Data

Manually trigger a refresh of the activities data:

/api/refresh.py

This endpoint logs in to Garmin Connect and fetches the latest activities.

Response Format

The response is a JSON object with the following structure:

{
  "schema_version": 1,
  "exported_at": "2023-05-17T14:25:10.123456Z",
  "activities": [
    { /* activity data */ },
    { /* activity data */ },
    ...
  ]
}
