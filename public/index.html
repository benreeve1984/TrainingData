Garmin Activities Exporter

About This Project

This service fetches activities from Garmin Connect and makes them available as JSON data. It automatically refreshes once per day at 20:00 UTC.

Data Access

Access your exported activities data at: /training.json

The data includes up to 2,000 of your most recent Garmin Connect activities.

API

Manually trigger a data refresh by sending a GET request to: /api/refresh.py

Note: This requires proper authentication credentials to be set in the environment variables.
